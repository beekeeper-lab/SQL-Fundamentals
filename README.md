# SQL Fundamentals

A structured, self-paced course for learning SQL from scratch using SQLite.

---

## Purpose

This course teaches the fundamentals of SQL -- from creating your first database to designing normalized schemas and writing optimized queries. By the end of this course, you will:

- Understand **relational databases** and how data is organized in tables, rows, and columns
- Write **DDL** (CREATE, ALTER, DROP), **DML** (INSERT, UPDATE, DELETE), and **DQL** (SELECT) statements
- Use **constraints** (PRIMARY KEY, FOREIGN KEY, NOT NULL, UNIQUE, CHECK, DEFAULT) to enforce data integrity
- Query data with **filtering** (WHERE), **sorting** (ORDER BY), **pagination** (LIMIT/OFFSET), and **deduplication** (DISTINCT)
- Summarize data with **aggregate functions** (COUNT, SUM, AVG, MIN, MAX) and **GROUP BY / HAVING**
- Combine data across tables using **joins** (INNER, LEFT, CROSS, self-joins)
- Build complex logic with **subqueries** and save reusable queries as **views**
- Safely modify data with **UPDATE**, **DELETE**, and **transactions**
- Optimize performance with **indexes** and protect against **SQL injection**
- Design clean, normalized schemas following **best practices**

### Who this is for

- Interns and junior developers learning SQL for the first time
- Anyone who has used spreadsheets and wants to understand databases
- Developers who need a structured, hands-on SQL foundation

### Tools

All exercises use **SQLite** -- zero configuration, no server needed. The entire database lives in a single file.

## Getting Started

1. **Install SQLite** (if not already installed):
   - Ubuntu/Debian: `sudo apt install sqlite3`
   - macOS: already included (or `brew install sqlite3`)
   - Arch Linux: `sudo pacman -S sqlite`

2. **Verify installation:**
   ```bash
   sqlite3 --version
   ```

3. **Start with Module 0** -- open `source/module-00-what-is-sql.md` and follow along

## Building the Course Website

This course uses the [Course Builder Guide](COURSE-BUILDER-GUIDE.md) pipeline to transform the Markdown source files into an interactive, self-contained course website with illustrations, narration, and pagination.

```bash
# Generate illustrations (requires GEMINI_API_KEY in .env):
# Ask Claude Code to generate images for each module

# Generate narration audio (requires ELEVENLABS_API_KEY in .env):
uv run --with elevenlabs python scripts/generate_narration.py --all

# Build the course HTML:
uv run --with markdown --with pygments python scripts/build_course.py

# Package for distribution:
uv run --with markdown --with pygments python scripts/deploy.py --version 1.0
```
